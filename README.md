# NeuraView

Smart AI-based residential security system using:
- Facial Recognition
- Gender Detection
- Web Application
- Android Application

## Features
- Real-time face detection
- Unknown person alert
- Mobile notification system
- Admin dashboard

## Tech Stack
- Python
- OpenCV
- TensorFlow / PyTorch
- Flask / Django
- Android (Java/Kotlin)

## Future Improvements
- Cloud deployment
- Live CCTV streaming
- Face database management
